# Svatební web a registrace hostů — přehled

Web slouží k informování hostů a hlavně k **registraci hostů přes formulář** (RSVP). Tady je postup, obsah a šablona formuláře.

## Checklist webu

- [ ] **Save the Date** — rozeslat hodně dopředu (než přijdou pozvánky), ať lidi s termínem počítají (zvlášť hosté, co cestují daleko).
- [ ] **Booknout doménu** (např. `jmeno-a-jmeno.cz`).
- [ ] **Upravit šablonu** (např. WordPress) — vzhled, fotky, texty.
- [ ] **Vytvořit skrytou sekci** s detaily pro hosty.
- [ ] **Zaheslovat skrytou sekci** a **heslo dát na pozvánky / oznámení**.
- [ ] **Naplnit web instrukcemi** (viz níže).
- [ ] **Vložit registrační formulář** (viz šablona otázek).
- [ ] **Pravidelně přenášet odpovědi z formuláře do `Hoste/pozadavky.md`.**
- [ ] **Měsíc před svatbou dotáhnout hosty, co neodpověděli.**

## Obsah webu (instrukce pro hosty)

- Program dne / harmonogram
- Místo konání + mapa
- Parkování + plánek
- Ubytování (kapacita, jak se přihlásit)
- Dress code
- Dárky / co si přejete
- Přezouvání, deky na louku, mazlíčci (ano/ne)
- Sdílené fotoalbum, kontakt na organizaci

## Registrační (RSVP) formulář — šablona otázek

Formulář, který hostům posíláte (např. Google Forms). Odpovědi pak přenášíte do `Hoste/pozadavky.md`.

1. **Časová značka** *(automaticky)*
2. **E-mailová adresa**
3. **Jméno**
4. **Mobilní číslo**
5. **Přijedu na svatbu [datum]:** ano / ne
6. **Stravovací dieta / alergie** *(lze zaškrtnout více možností:* bez omezení / vegetariánská / veganská / bezlepková / bezlaktózová / bez konkrétní suroviny / jiné*)*
7. **Přípitek chci:** s alkoholem / nealko
8. **Budu chtít přespat v areálu z [noc] na [noc]:** ano / ne
9. **Pokud nebudete přespávat na místě, budete chtít zajistit odvoz jinam?**
10. **Přijedete s dětmi?** *(počet + věk dětí)*
11. **Zdravotní nebo jiná omezení**
12. **Chcete nám sdělit ještě něco?**

> Tip: otázky 6, 8, 10 a 11 jdou rovnou do plánování (diety → menu; přespání → lůžka a snídaně; děti → kapacita a hlídání). Drž si u každé odpovědi **jméno hosta**.
