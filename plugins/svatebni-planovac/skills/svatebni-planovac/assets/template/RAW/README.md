# RAW — surové podklady

Sem se ukládá **vše, co dorazí, v původní podobě**: e-maily, zprávy, nabídky, fotky, scany smluv, přepisy hlasovek, poznámky ze schůzek.

## Pravidla
- **Nic se tu needituje.** Slouží jako archiv a důkaz, co bylo domluveno.
- Pojmenování: `RRRR-MM-DD_kratky-popis.md` (např. `2026-05-08_zapis-ze-schuzky-misto.md`).
- Členění do podsložek podle oblasti (Misto, Catering, Dort, Hudba, Matrika, Oddavajici, Fotograf, Kvetiny, Hoste, Koordinace, Ostatni).
- Klíčové info z podkladu se vždy přepíše do příslušného `Dodavatele/<dodavatel>/prehled.md` (nebo `Hoste/`, `Koordinace/`) a úkoly do `TODO.md`.
